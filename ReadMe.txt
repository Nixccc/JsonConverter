Version 1.0

1. Put your excel file in the folder "EXCELTOJSON"

2. Enable developer options:
	  (File -> Options -> Customize the Ribbon -> Customize the Ribbon -> Main Tabs -> Developer)

3. Import files (alt + f11 -> file -> import file): 
	  JsonConverter.bas
	  Moudule1.bas
	  Moudule2.bas
	  Moudule3.bas
	  Moudule4.bas
	  Moudule5.bas
	  Moudule7.bas
  	  Moudule8.bas
	  UserForm1.frm


4. Create a button 
	  (Developer -> Insert -> Form Controls -> Button)

5. Assign the Macro OpenSesame
	  (Right click on the button -> Assign macro -> OpenSesame)

6. Start the converter (Click on the button) 

Important: 
           Remember to put the engine row at 4 as it searches for engines on the fourth row
	   Remember to not delete the folder Json
